## Description
 
This API includes user management features, such as creating, reading, updating, and changing the status of users.
 
## Required
 
Please install the following before running the application:
 
- [Docker](https://docs.docker.com/get-docker/)
- [Docker Compose](https://docs.docker.com/compose/install/)
- [Git] (https://git-scm.com/downloads)
- [Node.js] (https://nodejs.org/en/download/)
- [Nest.js] (https://nestjs.com/)
 
 
## Installation
 
```bash
$ npm install
```
 
## Running the app
 
This application runs on Docker. You don't need to create any database instances. The database and database connection are already fully set up. Please proceed by following the commands below:
 
```bash

$ docker-compose -f ./db.docker-compose.yml up -d

# watch mode

$ npm run start:dev

# static mode

$ npm run start

```
